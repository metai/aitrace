.. _glossary:

名词术语
========

.. glossary::
   :sorted:

   Crisp Set
      明确集合 ( `Crisp Set <http://en.volupedia.org/wiki/Crisp_set>`_ ) 指经典集合, 主要强调元素是否属于集合是确定的.

   Fuzzy Set
      模糊集合 ( `Fuzzy Set <http://en.volupedia.org/wiki/Fuzzy_set>`_ ) 与经典集合不同, 主要强调元素是否属于集合是不确定的, 用隶属函数与隶属度来表示.

   Rough Set
      粗糙集合 ( `Rough Set <http://en.volupedia.org/wiki/Rough_set>`_ ) 与经典集合不同, 主要强调元素是否属于集合是不确定的, 用两个上下隶属函数与隶属度来表示.
