.. _glossary:

名词术语
========

.. glossary::
   :sorted:


   Fuzzy Mathematics
      模糊数学 ( `Fuzzy Mathematics <http://en.volupedia.org/wiki/Fuzzy_mathematics>`_ )

   Crisp Set
      明确集合 ( `Crisp Set <http://en.volupedia.org/wiki/Crisp_set>`_ ) 指经典集合, 主要强调元素是否属于集合是确定的.

   Fuzzy Set
      模糊集合 ( `Fuzzy Set <http://en.volupedia.org/wiki/Fuzzy_set>`_ ) 与经典集合不同, 主要强调元素是否属于集合是不确定的, 用隶属函数与隶属度来表示.

   Rough Set
      粗糙集合 ( `Rough Set <http://en.volupedia.org/wiki/Rough_set>`_ ) 与经典集合不同, 主要强调元素是否属于集合是不确定的, 用两个上下隶属函数与隶属度来表示.

   Fuzzy Logic
      模糊逻辑 ( `Fuzzy Logic <http://en.volupedia.org/wiki/Fuzzy_logic>`_ )

   Fuzzy System
      模糊系统 ( `Fuzzy System <http://en.volupedia.org/wiki/Fuzzy_system>`_ )

   Fuzzy Subalgebra
      模糊子代数 ( `Fuzzy Subalgebra <http://en.volupedia.org/wiki/Fuzzy_subalgebra>`_ )

   Neuro Fuzzy
      神经模糊 ( `Neuro Fuzzy <http://en.volupedia.org/wiki/Neuro-fuzzy>`_ )

   Membership Function
      隶属函数 ( `Membership Function <http://en.volupedia.org/wiki/Membership_function_(mathematics)>`_ ), 参见 :ref:`Section-MembershipFunction`

   Membership Grade
      隶属度 ( `Membership Grade <http://en.volupedia.org/wiki/Membership_grade>`_ ), 参见 :ref:`Section-MembershipFunction`

   Membership Degree
      隶属度 ( `Membership Degree <http://en.volupedia.org/wiki/Membership_degree>`_ ), 参见 :ref:`Section-MembershipFunction`

   Indicator Function
      示性函数 ( `Indicator Function <http://en.volupedia.org/wiki/Indicator_function>`_ ), 参见 :ref:`SubsubSection-SetCharacteristicFunction`
