Glade用户界面设计器
=====================


简介
----------------


`Glade <https://glade.gnome.org/>`_ 是一个开源的用户界面设计软件(User Interface Designer), 适用于包括 C, C++, C#, Vala, Java, Perl, Python 等语言在内的大量语言.



安装配置
---------------

Ubuntu
~~~~~~~~~~


``sudo apt install glade libglade2-dev``
