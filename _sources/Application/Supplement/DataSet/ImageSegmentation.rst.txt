图像分割数据集
===========================


视网膜图像
-----------------


- `DRIVE <http://www.isi.uu.nl/Research/Databases/DRIVE/index.php>`_
- `STARE <http://cecas.clemson.edu/~ahoover/stare/>`_ : STructured Analysis of the Retina
- `Tortuosity <http://bioimlab.dei.unipd.it/Retinal%20Vessel%20Tortuosity.htm>`_ : Retinal Vessel Tortuosity Data Set
- `HRF <http://www5.cs.fau.de/research/data/fundus-images/>`_ : automatic segmentation on retinal fundus images
- `retina dataset <https://github.com/yiweichen04/retina_dataset>`_ : Retina dataset containing 1) normal 2) cataract 3) glaucoma 4) retina disease
